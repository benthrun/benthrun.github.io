Dynamic Mode Decomposition Project

The final project in my data driven modeling course focused on the methods of Dynamic Mode Decomposition (DMD). I chose DMD to be focus of the project due to a suggestion from a friend. DMD is very popular withing the fluid dynamics community, and the special ability of this method allows for the user to reconstruct data and ability to predict future time states with past data.

The problem I chose to analyze took a Poisson Equation solver from my CFD course. The Poisson Equation solver determines convergence based on a number of iterations. Taking all the data from the solver I was able to construct one large matrix which contained all the information useful for the DMD procedure.

Using the DMD techniques I learned, I was able to reconstruct the data and successfully look into the future which illustrated the benefits of DMD and provided an interesting solution to this problem.
