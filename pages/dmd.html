Dynamic Mode Decomposition Project

For my graduate data driven modeling course final project, I was given the choice to explore a method discussed in class that I was interested in and wanted to explore further. I specifically chose the methods of Dynamic Mode Decomposition (DMD) to be the focus of the project. One of my interests is fluid mechanics and DMD is very popular withing the fluid dynamics community, the DMD method allows for the ability to reconstruct data and the abilty to predict future time states with past data. The problem I chose to analyze took a Poisson Equation solver from my CFD course. The Poisson Equation Solver determines convergence based on a number of iterations. Taking all the data from the solver I was able to construct one large matrix which contained all the information useful for the DMD procedure. Using the DMD techniques I learned, I was able to reconstruct the data and succedfully look into the future which illustrated the benefits of DMD and provided an interesting solution to this problem.
